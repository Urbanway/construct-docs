Response

A response is an output produced by MVC controller.

To output from a controller, you have either to return a string or \Construct\Response object. There are following extended types:

  • \Construct\Response\Json
  • \Construct\Response\Layout
  • \Construct\Response\PageNotFound
  • \Construct\Response\Redirect

You can implement your own reponse types by extending \Construct\Response class.

A controller should never echo output by itself.

Output a string

 This will render default theme layout with retuned string as a content for main block.

return 'Hello world';

Output in JSON format

 Mostly used for AJAX interactions

return new \Construct\Response\Json( array('exampleMessage' => 'Hello world'));

Display 404 Page not found message

 This will display error 404 page with correct headers.

return new \Construct\Response\PageNotFound();

Redirect a browser to some other page

return new \Construct\Response\Redirect('http://www.example.com');
